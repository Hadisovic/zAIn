## 🚀 Overview

**zAIn** merges advanced agentic AI capabilities with a desktop interface powered by Tauri, React 19, and local/remote LLM orchestration. 

Built on top of the **Sesame** core, it provides deep integration with your workspace, custom session tools, and state management.
